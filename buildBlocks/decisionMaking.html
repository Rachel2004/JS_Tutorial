Decision Making

Like other languages, JavaScript also provides many decision-making statements:
Statements Description Syntax
if "if" is used to check for a condition and whether it's true or not. The condition could be any expression that returns true or false. When the condition satisfies then the statement following the "if" statement is executed. if(condition){
statements...
}
else else statements are used with if statements. When the "if" condition fails then the "else" statement is executed. if(condition){
statements
}
else{
statements
}
else if Suppose there is a variety of conditions that you want to check. You can use multiple "if" statements to do this task. All the "if" conditions will be checked one by one. But what if you want that once a condition satisfies then don't perform further conditional checks? At this point "else if" statement is what you need. if(condition){
statements
}
else if(condition){
statements
}
else{
statements
}
switch case Switch statements are used when there are multiple conditions to check and when the conditions are values instead of ranges. var name;
switch(name)
{
case value-1: statements; break;
case value-2: statements; break;
case value-3: statements; break;
............
case value-n: statements; break;
default: statements; break;
}
ternary opretor Ternary operator is preferred by many programmers. It contains 3 operands that's why the name is "ternary operator". If the condition goes right then the statement before the colon is executed and if false then the statement after the colon is executed. Multiple statements are not allowed. But you can execute multiple statements by making a function and calling the function if the condition goes true or false accordingly. condition? if-true-execute-this : if-false-execute-this;
